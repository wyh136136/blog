# 行政组织理论



### 行政组织理论的研究意义

```
1、 研究行政组织理论是为了建立具有中国特色的行政组织理论体系.
2、 研究行政组织理论是实现行政组织科学化、 民主化法治化、 和现代化的需要,是社会健康发展的需要技术和管理是社会前进的动力.是推动现代生产力发展的两个轮子.
3、研究行政组织理论是我国当前行政组织改革的需要。它可以为行政组织改革的实践提供理论上的指导。行政组织改革是我国当前开展的政治体制改革、经济体制改革中的一项重要内容。也是政治
体制和经济体制改革继续深入进行的必要条件。
4、研究行政组织理论是优化行政工作人员的素质、提高其管理水平和工作能力的需要。
```

## 行政组织的演变

### 我国古代行政组织的特点：

```
1、强化中央集权。集国家最高权力为一人之手，是我国古代行政组织发展的主流。
2、中国古代行政组织的演变具有前后因袭，循序渐进的特点
3、中国古代行政组织结构严谨，分工精细，层级节制，结构统一
4、中国古代的行政组织具有强烈的宗法色彩
```

### 简述雅典行政组织的特点

```
1、雅典的行政组织具有原始的民主行政特征；
2、强调依法行政
3、行政官员非专业化、非职业化、无报酬
4、行政职能简单
```

### 中华民国行政组织的特点

```

1、中华民国的行政组织在其创立初期，接受了西方三权分立与民主共和国思想，
2、孙中山提出的政权和治权相区别，在国民大会下实行立法、行政、司法、考试、监察五权并立的思想，有历史的进步意义。
3、从法律上确定了国民党一党转正的政治体制，国民党中央执行委员会是权力核心。
4、在中央与地方的关系上，国民政府没有也不可能实现孙中山的均权制主张，而是中央集权制与地方军阀割据并存。
```

## 科层制组织理论

### 简述科层制组织的实践困境

```

1、形式主义导致的效力悖论
2、压抑个性导致的创新缺失
3、本位主义导致合作意识缺乏
4、文牍主义导致的积极性不足
5、权力垄断导致的反民主性
```

### 科层制组织的基本特征

```
1、专业化：专业化原则既适用于组织中的个人，也适用于组织单位或分工关系
2、层级化：又称等级化。行政人员和机构按照层级原则进行管理。
3、规则化：又称法制化、制度化
4、非人格化：处理公务时只考虑合法性、合理性及有效性
5、技术化：专业化、层级化的层级制要求管理者必须训练有素
```

## 行政组织结构

### 简述合理的行政组织结构的功能

```
1、合理的行政组织结构能有效的满足行政组织目标的需要；
2、有利于稳定工作人员的情绪，调动工作人员的积极性；
3、能使组织保持良好的沟通关系，是构成行政信息沟通的主要渠道
4、是提高微观和宏观行政效率的前提条件
5、有助于推动行政方式的创新
```

### 简述合理的行政组织结构的标准

```
1、任务与组织平衡
2、各个组织与人员之间按照比例进行配置
3、分工明确、合作良好行政组织结构的实质是以职能为内容进行分工
4、适应环境，具有弹性
```

### 简述行政组织横向分工的重要性

```
1、适应各项社会事务管理的需要
2、适应行政管理专业化、技术化的需要
3、适应行政管理综合协调、宏观管理的需要
4、适应行政管理程序的需要
```

### 简述行政组织横向结构的优点

```
1、职能不同，分工管理，有利于精通业务
2、从上到下形成条条，有利于统一管理
3、各个部门分工负责，有利于行政首长集中精力考虑全局
```

### 影响行政组织结构的主要因素

```
战略与结构、规模与结构、技术、任务与结构
```

## 行政组织体系

### 行政组织体制中分离制的优缺点

```
优点：
1、领导权力分散并相互牵制；可防止一级行政首长的独裁专断；
2、实行二元化领导，既有利于上级方针政策的有效贯彻，又有利于政府因地制宜的加以实施


缺点：
1、权力分散，易导致多头指挥，政令冲突，使得任何一个领导机关均指挥不灵；
2、政出多门，导致下级无所适从，或利用矛盾为己所用
```

### 简述分权制的优点

```
1、能够因地制宜，灵活机动的处理事务；
2、分权分工，可导致上级独断专行
3、符合民主原则
4、可以满足地方对公共物品和服务的差异化需求；
5、有利于监督
```

### 简述委员会制的优点

```
1、能集思广益，容纳各方意见。集中众人智慧，考虑问题较全面。
2、能承担集体责任，有利于协调一致的推行计划
3、互相监督，不易徇私舞弊
```

### 简述市经理制的优点

```
1、有一个统一而高度负责的行政首长，便于统一指挥，提高行政工作效率。
2、行政首长由市政专家担任，有利于提高城市管理的科学化水平
3、市经理对民选的市委员会负责，市委员会对选民负责，使专家的特长与选民的意志得以较好的结合
```

### 以行政活动程序为标准，可以将行政组织分为哪些类？

```
决策机关、执行机关、咨询机关、检察机关
```

## 行政组织的设置与自身管理

### 简述我国行政组织设置的效能原则

```
1、机构设置和人员编制要尽量精简
2、层级幅度要适中
3、进行专业化分工
4、简化办事程序
```

### 联系实际，概述行政组织规制性管理的优缺点

```
优点：
	强调理性化管理，要求在行政组织内部进行专业分工，层级节制和严格规范。
	注重组织的标准化、工作秩序化、管理规范化，目的是排除人的情感因素对组织运行的影响，尽可能消除偶然性。
缺点：
	造就了一种刚性的官僚系统，缺乏灵活性和主动精神，采用标准化的管理和服务，忽视了人的查差异性和多样性；
	排除了竞争的必要与可能，组织成员照章办事，墨守成规，条块分隔易产生问题，导致机构膨胀，职能交叉重叠
```

### 简述行为科学时期的行政组织设置原则

```
以人为本的原则；人与组织平衡的原则；参与、沟通、激励的原则；正式组织与非正式组织互动的原则
```

### 在行政组织的规制化管理中，管理规范化的主要内容有哪些？

```
1、组织成员的关系完全以理性准则为指导，坚决排斥一切人为的、情感的、偶然的因素，尤其的排斥不良人际关系的影响。
2、对所有人都一视同仁，使组织成员具有平等感和公平感
3、明确规定各成员的职权范围和协作形式，使各个成员正确行使职权，减少摩擦和冲突。
4、精确计算组织成员的工作成果，以业绩为依据进行奖惩，公平、客观、合理的处理问题
```

### 简述行政组织设置原则的作用

```
有助于设计与建立科学、合理的行政组织机构
有助于指导与规范行政组织的日常管理
有助于改革与完善行政组织
```

### 简述创建行政组织良好人际关系的原则与方法

```
平等民主、利益共享、意见沟通、激励强化
```

### 简述我国行政组织设置的职能决定原则

```
根据行政组织职能设置相应的机构
围绕组织只能确定组织结构

## 行政组织改革

### 结合当代英国行政组织改革的主要内容，试述其特点

```
内容：
1、重新界定和优化行政组织职能，将部分职能市场化；
2、以加强中央宏观调控权为宗旨，调整中央与地方行政组织之间的关系
3、以效率为中心，改革行政组织的内部管理体制
4、重视信息技术对行政改革的支持
5、改革人事制度
6、增强政府的回应性，优化公共服务

特点：
1、英国行政组织改革非常注重学习，借鉴私营企业的管理经验；
2、改革注重循序渐进
3、改革方案具有很强的针对性和操作性
```

### 试试当代中国行政组织改革的背景及主要经验

```
背景：
1、社会主义市场经济体制的逐步建立，要求行政组织进行相应的改革
2、民主化政治体制的改革为我国行政组织的改革提供了广阔的空间
3、传统的行政组织体制，使行政效率低下，导致社会公众不满，为行政组织改革提供了社会动力
4、国外行政改革的理论和实践给我国行政组织改革提供了有益的经验和借鉴
5、知识经济时代的到来，给我国行政组织改革带来了机遇和挑战


主要经验：
1、坚持以适应社会主义市场经济体制为改革的目标，把转变政府职能作为机构改革的关键，行政机构改革，提出了以转变职能为中心。
2、坚持精简、统一、效能的原则，把精兵简政和优化政府组织结构作为机构改革的重要任务。
3、坚持渐进、稳妥的改革方针，坚持统一领导，分级负责，分步实施，从实际出发，因地制宜的进行改革
4、坚持机构改革与干部人事制度改革相结合，制定配套的政策措施，妥善安排分流人员，优化干部队伍结构
```

### 简述美国行政改革中精简机构的主要措施

```
1、合并机构、精简人员、减少财政支出
2、建立创新基金
3、建立电子政府
```

### 当代日本行政组织改革的主要内容，并简单评述

```
内容：
1、放松国家规制，调整官民关系，缩减政府职能机构
2、推动地方分权，重新调整中央与地方的关系
3、建立防止行政机构膨胀的机制
4、建立公务员交流制度，纠正条条行政的弊端。

评述：
1、行政改革过程中强调民主参与和公开透明；
2、注重依法改革；
3、根据形势的发展，积极主动的进行行政改革
```

### 试述当代美国行政组织改革中政府职能市场化的主要做法。

```
所谓的政府职能市场化，是指在对政府职能重新界定后，将原来一部分属于政府的职能交给社会组织或民间团体去承担，由市场机制发挥作用。

美国政府职能市场化的主要做法：
1、实行公共服务民营化，政府不再垄断公共服务的供给，政府公营的公共服务部门私有化。
2、实行政府业务合同化，对那些仍然属于政府职能范围内的事项通过合同的方式“外包”
```

### 美国行政组织改革的背景及主要内容

```
背景：
1、第二次世界大战后到20世纪70年代末，美国政府行政职能的扩张引起公众的不满，
2、美国政府规制过多过滥，抑制了组织及其工作人员的积极性和创造性
3、信息技术的发展为总政组织的变革创造了有利条件
4、国际环境的变化也促使美国行政组织出现了一些新变化。

内容：
    政府职能市场化；
    放松规制；
    放权与分权并行；
    精简机构。
```

## 真题

### 中华民国的行政组织特点

```
